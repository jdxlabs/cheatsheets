# Terraform basic commands

```bash

terraform init

terraform validate

terraform plan

terraform apply

terrafom destroy

also you can use tfenv to handle several versions

```


## Alias to clean .terraform files in a folder

```bash
alias clean_tf='find . -name ".terraform" -type d -exec rm -rf {} \;'
```
