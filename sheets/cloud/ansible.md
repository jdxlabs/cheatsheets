# Ansible

## Ping
```bash
ansible -m ping all
ansible -m ping all -b
```

## Initialize a new role
```bash
ansible-galaxy init <myrole>
```

## Install a specific version
```bash
sudo pip install --upgrade ansible==2.4.4
```
