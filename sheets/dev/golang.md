# Go language

## Create a Go path
```bash
apt install golang-go
mkdir -p $HOME/gocode/bin
export GOPATH=$HOME/gocode
export PATH=$PATH:$GOPATH/bin
```

## Get a lib
```bash
go get -v <my_lib>
```
