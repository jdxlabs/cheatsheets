# Claude


To save Token, configure it with a low effort model by default. You can change it later in the settings.
```
# ~/.claude/settings.json
{
    "model": "claude-sonnet-4-6",
    "effortLevel": "low"
}
```

It is also possible to apply with commands : /models & /effort low.


Pertinent first questions to ask Claude when coming on a new repository :
* Give me an overview of the repository and its purpose.
* What are the main components of the codebase?
* Are there any specific coding conventions or best practices I should follow?
